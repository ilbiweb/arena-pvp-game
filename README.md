# 🐍 Arena PvP - Snake Battle on Solana

Juego PvP donde dos jugadores apuestan SOL y compiten en Snake. El ganador recibe el 95% del pozo.

## Requisitos previos
- Navegador con [Phantom Wallet](https://phantom.app/) instalada
- SOL en Devnet (para pruebas) o Mainnet (producción)
