# 🐍 Arena PvP - Snake Battle en Solana

Juego PvP donde dos jugadores apuestan SOL y compiten en Snake.  
**El ganador recibe el 95% del pozo. El 5% es comisión para el desarrollador.**

## 🚀 Jugar ahora

🔗 **https://ilbiweb.github.io/arena-pvp-game/**

## 📱 Requisitos

- [Phantom Wallet](https://phantom.app/) (PC o Android)
- Red: **Devnet** (para pruebas) o **Mainnet** (dinero real)

## 🎮 Cómo jugar

1. Conecta tu wallet Phantom
2. Cambia a la red **Devnet** (Settings → Network → Devnet)
3. Obtén SOL de prueba en https://faucet.solana.com
4. Crea una sala con tu apuesta (mínimo 0.1 SOL)
5. Espera a que otro jugador se una
6. Presiona "Iniciar Partida"
7. Usa las **flechas del teclado** para controlar la serpiente
8. ¡Gana y reclama tu premio!

## ⚙️ Tecnologías

- **Blockchain**: Solana (Devnet)
- **Smart Contract**: Anchor (Rust)
- **Frontend**: HTML5, CSS3, JavaScript
- **Wallet**: Phantom
